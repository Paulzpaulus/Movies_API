# Movies API

Simple Python project to manage movies using SQLite and OMDb API.

## Features
- Add, delete, update movies
- Store year, rating, poster
- Search and stats
- Generate static website (index.html)

## Setup
```
pip install -r requirements.txt
```

Create `.env`:
```
OMDB_API_KEY=your_key
```

## Run
```
python main.py
```
